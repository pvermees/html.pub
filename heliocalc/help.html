OPEN: opens input data and settings, stored in a .json format
SAVE: saves input data and settings in a .json format
IMPORT: loads input data (but no standard solution and spike settings) in a .csv format
EXPORT: saves input data in a .csv format which can be opened in a spreadsheet format
HELIOPLOT: saves input data in a .csv format for plotting in HelioPlot

The source code can be downloaded from GitHub. Doing so allows the calculator to be run offline by opening index.html in a browser.To use the OPEN, SAVE, IMPORT, EXPORT and HELIOPLOT functions offline, you need to simulate a webserver on a local system. The easiest way to do this is by installing python on your system, opening a text terminal, navigating to the directory containing index.html, and typing python -m SimpleHTTPServer'. The calculator can then be accessed through your browser at http://localhost:8000
