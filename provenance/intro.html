provenance is a software package in the R programming environment, which aims to facilitate the visualisation and interpretation of `Big Data' in the context of sedimentary provenance analysis. After installing R from http://r-project.org, provenance can be installed by typing 
install.packages('provenance')
Once installed, the package can be loaded by typing 
library(provenance)
There are TWO WAYS to run provenance:
  1. The package comes with a query-based user interface which does not require any knowledge of the R programming language.
  2. Experienced users can access all the functionality from the command line using the provenance API.
Further details about these two modes are provided in two sets of tutorials, which can be accessed from the tabs above, and in the Sedimentary Geology paper. The tutorials use an example dataset from Namibia, comprising the following files:

Downloads:
- DZ.csv - detrital zircon ages
- DZerr.csv - standard errors of the detrital zircon ages
- HM.csv - heavy mineral counts
- PT.csv - bulk petrography
- PTHM.csv - combined petrography and heavy minerals
- Major.csv - major element composition
- Trace.csv - trace element composition
- densities.csv  - densities of common minerals and rock fragments
- endmembers.csv  - mineralogy of compositional end members

Citeable reference:
Vermeesch, P., Resentini, A. and Garzanti, E., 2016. An R package for statistical provenance analysis. Sedimentary Geology, doi:10.1016/j.sedgeo.2016.01.009.
