Pieter's software

  1. CosmoCalc
    • CosmoCalc.xla: an Excel add-in for cosmogenic nuclide calculations.
    • CosmoTest.xls: spreadsheet with test data
  2. RadialPlotter
    • RadialPlotter.jar: a Java application for fission track, luminescence and other radial plots.
    • FTdata.csv: example of fission track input data.
    • OSLdata.csv: example of luminescence dose rate input data.
  3. DensityPlotter
    • DensityPlotter.jar: a Java application for Kernel Density Estimation (KDE) of detrital age distributions.
    • UPbdata.csv: example of detrital zircon UPb age input data.
  4. HelioPlot
    • HelioPlot.jar: a Java application to plot U-Th-(Sm)-He data on ternary diagrams and logratio plots.
    • UThHe.csv: example input file with U-Th-He data.
    • UThSmHe.csv: example input file with U-Th-Sm-He data.
